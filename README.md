# A Toca do Hobbit — Experiência Web Interativa

Este projeto é uma página web interativa inspirada em ambientes de fantasia medieval. A proposta é criar uma experiência visual imersiva baseada no universo de hobbits, utilizando animações, efeitos de transição e integração com visualização 3D.

A aplicação começa com uma cena inicial que apresenta uma casa hobbit em um ambiente visual. Ao interagir com o botão de entrada, o usuário passa por uma transição animada que revela o conteúdo principal da página. O site utiliza elementos visuais como runas animadas, iluminação temática e efeitos de rolagem para criar uma atmosfera semelhante a um cenário de fantasia.

O conteúdo da página apresenta seções que descrevem elementos de uma toca hobbit, como biblioteca, cozinha e paisagem ao redor, utilizando cartões estilizados e citações inspiradas em histórias de aventura. A interface busca simular um ambiente acolhedor e mágico, combinando tipografia temática, cores quentes e efeitos de iluminação.

Além da página principal, o projeto também inclui uma página adicional com uma visualização 3D interativa incorporada através de um iframe. Essa página permite que o usuário explore um ambiente interno de uma casa hobbit utilizando um modelo hospedado externamente.

## Tecnologias utilizadas

- HTML5 para estrutura da interface
- CSS3 para layout, animações e estilização temática
- JavaScript para interatividade e efeitos dinâmicos
- Google Fonts para tipografia personalizada
- Sketchfab para visualização de modelos 3D incorporados

## Funcionalidades principais

- Tela inicial com animação de entrada
- Transição visual entre a cena inicial e o conteúdo do site
- Efeito de decodificação de runas no título principal
- Carrossel animado de símbolos rúnicos
- Revelação de elementos conforme o usuário rola a página
- Layout responsivo com seções organizadas em cartões
- Página adicional com visualização 3D interativa

## Execução do projeto

Para visualizar o projeto, basta abrir o arquivo `index.html` em qualquer navegador moderno. Não é necessário instalar dependências ou utilizar servidor, pois todo o projeto utiliza tecnologias web nativas.

## Objetivo do projeto

O objetivo deste projeto é explorar conceitos de design visual, animações em CSS, manipulação do DOM com JavaScript e criação de experiências web interativas. O foco principal é demonstrar criatividade na interface e no uso de efeitos visuais para criar uma ambientação temática.

Este projeto também pode servir como estudo de animações front-end, integração de conteúdo 3D em páginas web e construção de layouts estilizados para portfólio ou projetos criativos.
